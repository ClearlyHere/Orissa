Router
in package
  • Application

Router.php : 24

Router takes in charge handling requests by the User through the URI. The router reads the URI and checks whether the route read from the URI exists or not.

The first path will always be Orissa as it is the website itself. The second path will always be a controller which will take in charge the actions of said route. The rest of the paths will be defined in the routesMap of the associated Controller.

For example, the route : /Orissa/Login/Logging

/ Orissa / Login / Logging ^ ^ ^ First path Second Path Third Path Website Controller Action for said Controller
